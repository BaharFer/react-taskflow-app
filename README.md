# React Todo App

A modern and responsive Todo application built with React.

## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Filter completed tasks
* LocalStorage support
* Responsive design
* Glassmorphism sidebar UI
* Component-based architecture

## Technologies Used

* React
* JavaScript (ES6+)
* CSS3
* Vite

## Project Structure

src/
├── components/
│ ├── Sidebar.jsx

│ ├── TodoList.jsx
│ └── TodoItem.jsx
│
├── App.jsx
├── App.css
├── index.css

## Installation

```bash
npm install
npm run dev
```
